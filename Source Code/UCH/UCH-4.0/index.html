Abstract

The Trace Universal Control Hub for Java (UCHj) is a Java implementation of the Universal Control Hub project.

UCHj is based on Java 1.6 or later. Aside from the source code, a (binary) UCH.war file is provided for running UCHj as servlet under Apache Tomcat, also an installer is provided for the Windows operating system.
