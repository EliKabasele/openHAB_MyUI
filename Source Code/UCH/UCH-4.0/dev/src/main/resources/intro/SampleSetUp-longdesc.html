Figure 1 shows sample setup of UCHj, Controllers and Target Devices.

  • Assume that all targets and controllers are connected via wired or wireless LAN.
  • UCHj is running on Desktop PC.
  • EZFetch is a Media Renderer and Buffalo Linkstation is a Media Server.
  • GC-100 is a master controller that provides IP gateway to control Serial and InfraRed devices. Denon DVD and Sony Bravia are connected with GC-100 via IR Cable.
  • iPhone, Windows Mobile and Windows Tablet PC can be used to run contollers like iPhone-Simple, iPhone-Standard, DragIt or PlayIt.
  • These controllers communicate with UCHj using URC-HTTP protocol.
  • UCHj communicates with targets like Media Server, Media Renderer and GC-100 using their native protocol.
