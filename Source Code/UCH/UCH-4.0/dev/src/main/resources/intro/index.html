Thank you for trying the Trace UCH Live CD.

Please make sure that you are connected to the Internet. This CD does not make any changes to your PC configuration.

You can now use your computer as a remote control for home devices and appliances. Sample setup.

Steps for using the software:

  1. Please make sure all the UPnP devices are connected.
  2. If you have GC-100 master controller, you can also remotely control infrared or serial devices attached to it. For configuring GC-100, please read the tutorial for GC-100 setup.

    Note: Please note that settings of GC100 master controller configuration are non-persistant i.e. on restarting the computers settings will not be saved.

  3. UCH will automatically connect to all UPnP Media Servers & Renderers, also infrared and serial devices configured through GC-100 master controller.
  4. Now Open user interface list to get the list of user interfaces like "iPhone Interface STANDARD", “iPhone Interface SIMPLE”, “DragIt”.
