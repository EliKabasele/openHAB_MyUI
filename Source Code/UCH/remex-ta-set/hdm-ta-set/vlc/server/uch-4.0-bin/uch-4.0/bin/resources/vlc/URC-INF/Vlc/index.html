1. Introduction

The following documents pertain to a tv-set as a URC target. They can be used as template or the final URC files for controllable tv-set.

2. Conformance

A device conforms to this specification, if either one of the following is true:

  • The device's user interface socket inherits from the template socket, as provided in section Master Documents.
  • The device's user interface socket description has all of the following characteristics:
    1. The socket description references the template socket through the Dublin Core 'source' element <dc:source>.
    2. The socket description contains all sets of the template socket in an unmodified form.
    3. The socket description contains all mandatory (non-optional) socket elements (variables, commands, notifications) of the template socket in an unmodified form. If it contains optional socket elements of the template socket, these are unmodified.
    4. The socket description contains all internal type and element definitions, as defined in the schema part of the template socket.

A conforming device may employ the target description, as provided in section Master Documents. It may do so in literal or modified form.

A conforming device may employ the grouping sheet, as provided in section Master Documents. It may do so in literal or modified form.

A conforming device may employ the resource sheet, as provided in section Master Documents. It may do so in literal or modified form.

3. Master Documents

The following URC master documents for tv-set are provided as attachments. They have been defined following the target requirements defined by [ISO/IEC 24752-1]

  • Target description Defined following normative reference [ISO/IEC 24752-4]
  • Socket description Defined following normative reference [ISO/IEC 24752-2]
  • Grouping sheet Defined following normative reference [ISO/IEC 24752-5]
  • Resource sheet for English resources for the target description Defined following normative reference [ISO/IEC 24752-5]
  • Resource sheet for English resources for the user interface description description Defined following normative reference [ISO/IEC 24752-5]
  • Resource sheet for English resources for the grouping sheet description Defined following normative reference [ISO/IEC 24752-5]
