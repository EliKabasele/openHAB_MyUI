MyUI-based implementation of an application to control coloured light bulbs via a UCH.

Setup
-----
- Copy all files and folders into a folder served by the UCH (e.g. bin/Webclient/myui/).
- Start the UCH (see instructions there).
- Point your chrome browser to the served index.html (e.g. http://localhost:8080/UCH/Webclient/myui/index.html).
