1. Introduction

Using coloured light is a powerful way to create impressive atmospheres. Here, the expression coloured light bulb is used as a synonym for any light source that can change its colour. The User Interface Socket Description for a coloured light bulb contains all elements to control the colour, brightness and saturation. All documents can be either used directly or can be extended.

2. Conformance

A service conforms to this specification, if either one of the following is true:

  • The service's user interface socket inherits from the template socket, as provided in section Master Documents.
  • The service's user interface socket description has all of the following characteristics:
    1. The socket description references the template socket through the Dublin Core 'source' element <dc:source>.
    2. The socket description contains all sets of the template socket in an unmodified form.
    3. The socket description contains all mandatory (non-optional) socket elements (variables, commands, notifications) of the template socket in an unmodified form. If it contains optional socket elements of the template socket, these are unmodified.
    4. The socket description contains all internal type and element definitions, as defined in the schema part of the template socket.

A conforming service may employ the target description, as provided in section Master Documents. It may do so in literal or modified form.

A conforming service may employ the resource sheets, as provided in section Master Documents. It may do so in literal or modified form.

3. Master Documents

The following URC master documents for Coloured Light Bulb are provided as attachments.

  • ColouredLightBulb.td - Target description for a coloured light bulb
  • ColouredLightBulbSocket.uis - Socket description for a coloured light bulb
  • ColouredLightBulb.de.rsheet - Resource sheet for German resources
  • ColouredLightBulb.en.rsheet - Resource sheet for English resources
  • ColouredLightBulbSocket.de.rsheet - Resource sheet for German resources
  • ColouredLightBulbSocket.en.rsheet - Resource sheet for English resources
